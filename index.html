Why is Software Testing important - if developers write perfect code, why do you need to test it?

Software testing is an essential part of the software development process. It helps to ensure that the software is of high quality, reliable, and meets user requirements.

Even if there was such thing as a perfect developer, testing can pick up on performance issues, ensure that the software is working reliably, and can even be used to guide and define the needs of a project when used in Test Driven Development.

Testing also helps to ensure that the software is accessible to everyone, and that it meets the needs of all users. It can also help to identify and fix bugs before they are released to the public.

Finally, and most importantly, the dopamine rush of seeing all that green text is just really, really great.
